# Lead Engine — Claude Skill

> Automated B2B lead generation pipeline powered by **DataForB2B** and **LinkupAPI**.

Give it your website. Get qualified leads in your inbox — automatically, every day.

---

## Install

**Claude Code / CLI:**
```bash
npx skills add Eliott-89/lead-engine
```

**Cowork (scheduled automation):**
1. Download the ZIP from **[dataforb2b.ai](https://dataforb2b.ai)** → Skills section
2. Cowork → **Skills → Create a skill → Upload a skill** → upload the ZIP directly

Then in Claude:

```
/lead-engine https://yourwebsite.com
```

That's it. Lead Engine will build your ICP, show you sample prospects, and guide you through the rest.

---

## How it works

**No setup required to get started.** Lead Engine shows you what it can do before asking you to connect anything.

1. **Demo** — Analyzes your website, builds your ICP, shows sample matching prospects
2. **Refine** — Adjust your ICP until it's exactly right
3. **Setup** — Connect DataForB2B + LinkupAPI (5 min, guided)
4. **Find leads** — 20 qualified prospects per day, automatically scored
5. **Outreach** — Choose from 3 message styles, send after connection accepted
6. **Automate** — Schedule daily runs, track everything in a CSV

## Safety

- Max 20 connection requests/day
- Connections sent without a note — message only after acceptance
- Stops automatically on platform checkpoint detection
- You approve every lead list and every message before sending
